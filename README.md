# Cafeteria Project ☕

This is a web system developed for the management of a coffee shop's products, enabling CRUD (Create, Read, Update, and Delete) operations.

## 🚀 About the Project
The project is structured to be a functional inventory/menu management application, using standard web technologies for the front-end and PHP for back-end data handling.

## 🛠️ Technologies Used
- **PHP**: Language used for server-side logic and data manipulation.
- **HTML5**: Page structure.
- **CSS3**: Styling and responsive design.
- **JavaScript**: Client-side interactivity.

## 📂 Project Structure
```
text
├── css/                # Style sheets (CSS)
├── img/                # Images and visual assets
├── js/                 # JavaScript files
├── src/                # Additional source code
├── admin.php           # Administrative panel
├── cadastrar-produto.php # Product registration functionality
├── editar-produto.php  # Product editing functionality
├── excluir-produto.php # Product deletion functionality
├── index.php           # System home page
├── login.html          # Authentication interface
└── README.md           # Project documentation
```
## ⚙️ Main Features
Authentication: System access via login.html.

### Product Management:

Listing: View all available products.

Registration: Add new items to the menu.

Editing: Update existing product information.

Deletion: Remove products from the system.

## 📋 How to Run
To run this project locally, you will need a PHP server environment (such as XAMPP, WAMP, or any environment with Apache/PHP installed):

Clone this repository into your server's root folder (e.g., htdocs in XAMPP).

Ensure that your server service (Apache) is running.

Access the project through your browser, usually at http://localhost/cafeteria/index.php.
